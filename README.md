# Full-Scale Operation - GMTK Jam 2024

![Cover](./media/cover.png)

Play the game [here](https://sundowns.itch.io/full-scale-operation)

## Controls

* [WASD/Arrows] Move
* [SPACE] Jump
* [SHIFT] Punch
* [E] Grab/Drop item
